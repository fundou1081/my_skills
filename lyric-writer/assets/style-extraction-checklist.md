# 风格提取检查清单 (Style Extraction Checklist)

> 提取完风格画像后, 用这个清单检查完整性。  
> 歌词生成后, 用对应的"自检清单" 验证质量。

---

## 软维度 (5 项 — 每次创作都问)

- [ ] **1. 核心情感** — 一句话说清
- [ ] **2. 内心需求** — 听众反应
- [ ] **3. 创作源头** — 灵感来源
- [ ] **4. 特殊技法** — 独特手法
- [ ] **5. 表达方式** — 直白 / 含蓄 / 呐喊

## 硬维度 (3 项 — 从风格库加载)

- [ ] **6. 押韵方案** — 密度 + 位置
- [ ] **7. 句式模板** — 主歌/副歌/桥段
- [ ] **8. 意象清单** — 10-20 个常用物象

## 代表作品 (3-5 首)

- [ ] 列出 3-5 首
- [ ] 每首附核心意象 + 1 句最戳的歌词

## 用户确认

- [ ] 用户对画像 **✅ 确认**
- [ ] 或 **✏️ 修改某些字段** (回 Step 2)
- [ ] 或 **❌ 换风格** (回 Step 1)

---

# 歌词自检清单 (Song Self-check)

> 歌词生成后用。

## 通用 (所有风格)

- [ ] 核心意象出现至少 3 次, 每次含义略不同
- [ ] 每段有具体画面感
- [ ] 朗读节奏自然, 无拗口
- [ ] 情感弧线完整
- [ ] 结尾避免说教, 用意象/动作收束
- [ ] 整体字数适合谱曲 (300-500 字)

## 风格化 (按风格补全)

- [ ] **古风**: 至少 3 句矛盾式金句, 时空折叠至少 1 处
- [ ] **流行**: hook 记忆点, 副歌朗朗上口
- [ ] **民谣**: 第一人称真实感, 时间线清晰
- [ ] **摇滚**: 力量感, 鼓点配合
- [ ] **R&B**: 律动 + 转音预留
- [ ] **说唱**: 押韵密度高, flow 切分
- [ ] **城市民谣**: 都市细节真实
- [ ] **校园民谣**: 青春感, 时间跳跃
- [ ] **电子**: hook 重复, drop 蓄势
- [ ] **中国风**: (待用户定义)
