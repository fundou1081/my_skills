# 押韵方案库 (Rhyme Schemes Library)

> 不同风格的押韵规则, 给 AI 决策时参考。

---

## 古风 (Gufeng)
- **主歌**: 押韵可宽松 (AABB / ABAB 均可)
- **副歌**: 严格 (对称七字句必须押韵)
- **桥段**: 四字短句, 节奏爆发
- **常用韵脚**: 遥/潮/烧/涛/桥/凋/消/刀/笑/了/到/晓/老/高/涛

## 流行 (POP)
- **主歌**: 押韵宽松 (允许单押)
- **副歌**: 严格双押 (AA / BB)
- **pre-chorus**: 蓄势, 可不押
- **常用韵脚**: 爱/来/开/在/海/白/外/排/败/坏/猜/拍

## 民谣 (Folk)
- **押韵宽松**, 甚至允许不押
- 强调叙事真实
- 偶尔关键句押韵即可
- **常用韵脚**: 路/树/雨/度/雾/步/住/诉/楚/书

## 摇滚 (Rock)
- 押韵次要, 节奏为主
- 桥段 / 副歌可大量重复
- **常用韵脚**: 火/我/破/活/锁/错/过/果/脱

## R&B / 灵魂
- 押韵宽松
- 强调转音 + 假声, 押韵为辅
- 偶尔 hook 押韵
- **常用韵脚**: you/me/tonight/right/light/feel/real

## 说唱 (Hip-hop)
- **押韵密度极高**
- 多押 / 双押 / 三押
- 强调 flow 切分
- **常用韵脚**: 任何, 关键是密度

## 城市民谣 (Urban Folk)
- 押韵宽松, 类似民谣
- 偶有金句押韵

## 校园民谣 (Campus Folk)
- 押韵宽松
- 4-7 字句交替

## 电子 (EDM)
- 押韵次要
- hook 重复为主
- 偶尔口号式押韵

## 决策流程

```
用户给风格 + 主题
   │
   ├─ 查表找该风格的押韵规则
   │
   ├─ 选韵脚 (基于主题情感 + 风格)
   │
   ├─ 套用密度 (主歌/副歌/桥段)
   │
   └─ 生成时持续检查押韵位置
```

**重要**: 押韵是为了**节奏和记忆点**, 不是为了炫技。说唱的"押韵密度极高" 是因为 flow 的需要, 不是为了押韵而押韵。
