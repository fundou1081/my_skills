# 古风 (Gufeng) 风格

> **状态**: ✅ 完整实现 (从原 SKILL.md 6 阶段迁移并重组为 8 维度格式)

## 风格定义

以中国传统文化为底蕴, 讲究意象美、韵律美、情感深沉的歌词风格。  
代表人物: 方文山、林夕 (部分)  
代表作品: 《归潮》《青花瓷》《东风破》《千里之外》

## 8 维度画像

### 1. 核心情感
- 思乡、怀旧、怅惘、放下、执念
- 情感弧线模板: "历经___，最终___"
- 核心矛盾: 越___越___ / 失去了才___ / 明明___却___

### 2. 内心需求
- **看破 → 放下 → 升华** 的情感层次
- 不直接说"我很孤独", 而是用空城/灯/月/潮 等意象暗示
- 追求"诗化"表达, 避免直白

### 3. 创作源头
**意象库**:
- **自然类**: 潮、风、雪、舟、灯、镜、剑、月、雁、枫
- **建筑类**: 城、楼、桥、塔、门、墙
- **抽象类**: 梦、痕、劫、忆、缘
- **情感对应**: 红袍、刀剑、空城、流萤、芳草

### 4. 特殊技法
- **时空折叠**:
  - 空间并置: "空城里听浪涛"
  - 时间叠合: "岁月里话今朝"
  - 虚实转换: "满天流萤化成了芳草"
- **矛盾修辞**: 越___越___ / 失去了才___ / 明明___却___

### 5. 表达方式
- 不直白, 用意象暗示
- 细节代替概括: ❌"我很孤独" → ✅"空城里听浪涛"
- 短句与长句交替, 制造节奏感
- 结尾避免说教, 用意象收束

### 6. 押韵方案
- 主歌: 押韵可宽松
- 副歌: 对称七字句, 押韵严格
- 桥段: 四字短句, 节奏感强

### 7. 句式模板
- **A1 (主歌1)**: 4 字 + 7 字 + 4 字 + 7 字
- **B (预副歌)**: 矛盾句对偶
- **C (副歌)**: 7 字对偶, 易传唱
- **D (桥段)**: 4 字短句, 节奏爆发

**标准结构**: A1 → B → C → A2 → B2 → C2 → D → C3 (升华副歌)

## 6 阶段创作流程 (古风专用)

### 第一阶段: 锚定核心意象与主题

```
🎯 第一阶段: 锚定核心意象与主题

要写一首好歌词, 先要找到一个能贯穿全篇的"意象"。

【可选意象库】
自然类: 潮、风、雪、舟、灯、镜、剑、月、雁、枫
建筑类: 城、楼、桥、塔、门、墙
抽象类: 梦、痕、劫、忆、缘

【请告诉我】
1. 你想用什么作为核心意象? (或告诉我情感主题, 我来推荐)
2. 用一句话概括情感弧线: "历经___, 最终___"
```

### 第二阶段: 构建歌词骨架

**字数节奏**:
- 主歌: 四字/五字 + 七字交替
- 副歌: 对称七字句, 便于传唱
- 桥段: 四字短句, 增强节奏

**示例输出**:
```
【骨架规划】

A1 段 (4 句):
- 四字: ___消___
- 七字: ___
- 四字: ___迢遥
- 七字: ___

B 段 (2 句矛盾句):
- 越___越___
- 半生___半生___

C 段 (4 句副歌):
- ___如___
- 与我___
- 自有___
- ___凭吊
```

### 第三阶段: 意象系统与时空编织

```
🎨 第三阶段: 搭建意象系统

围绕你的核心意象, 列出相关物象:

【意象群搭建】
核心意象: 潮
├── 关联实景: 江、舟、岸、浪、潮汐、月
├── 情感对应: 红袍、刀剑、空城、流萤、芳草
└── 时空手法: 空间并置、时间叠合、虚实转换
```

### 第四阶段: 文字填充

**技巧**:
1. **矛盾修辞**
   ```
   越忘不了越看不明了
   越放开手越汹涌灼烧
   失去了才逍遥
   ```
2. **细节代替概括**
   - ❌ "我很孤独"
   - ✅ "空城里听浪涛"
3. **节奏控制**: 朗读调整长短句交替, 关键位置押韵

### 第五阶段: 情感弧线打磨

```
[起] 状态描写: 流云消、城门睡倒
[承] 挣扎深化: 山海誓、醉时歌
[转] 看破放下: 越放开手、红尘喧嚣
[合] 升华归途: 归途再爱一遭
```

### 第六阶段: 自我检验

**检查清单**:
- [ ] 核心意象出现至少 3 次, 每次含义略不同
- [ ] 至少有 3 句矛盾式金句
- [ ] 每段有具体画面感
- [ ] 朗读节奏自然, 无拗口
- [ ] 情感弧线完整 (低-高-低-升华)
- [ ] 结尾避免说教, 用意象收束
- [ ] 整体字数适合谱曲 (300-500 字)

## 代表作品分析

### 《归潮》(方法论来源)
- 核心意象: 潮 (出现 4 次)
- 意象群: 流云、晚潮、江潮、浪涛
- 情感弧线:
  - 起: 漂泊感 (流云消、路迢遥)
  - 承: 执念 (山海誓、醉时歌)
  - 转: 看破 (越放开手、红尘喧嚣)
  - 合: 升华 (归途再爱一遭)
- 亮点: 矛盾式金句密集、虚实交替恰当、结尾余韵悠长

### 《青花瓷》(方文山)
- 核心意象: 青花瓷
- 特殊技法: 物象拟人 ("瓶口描绘的牡丹一如你初妆")
- 情感: 含蓄婉约, 留白

## 与其他风格的区别

- **vs 中国风 (chinese-style)**: 古风偏传统/文言, 中国风偏流行化 (待用户定义)
- **vs 流行 (pop)**: 古风重意象意境, 流行重旋律记忆
- **vs 民谣 (folk)**: 古风重韵律美, 民谣重叙事真实
