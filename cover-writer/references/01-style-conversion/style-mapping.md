# 风格映射表 (Style Mapping)

> 8 大目标风格 × 配器/节奏/调性/描述 完整映射。

## 1. 电子 (Electronic)

### EDM
- **配器**: 808 bass, 合成器 lead, 强 kick, snare, hi-hat
- **节奏**: 128 BPM, 4/4 拍, 强劲 drop
- **调性**: 大调为主, 副歌升 key
- **人声**: 男/女声, 可加 autotune
- **描述模板**: "EDM, 强劲 drop, 合成器, 快节奏, 适合夜店"

### Synthwave
- **配器**: 复古合成器, 模拟鼓机, 锯齿波 lead
- **节奏**: 110-120 BPM
- **调性**: 小调, 复古感
- **人声**: 男声, 略带混响
- **描述模板**: "Synthwave, 复古合成器, 80 年代风, 略带失真"

### Lo-fi
- **配器**: 钢琴, 低保真鼓, 黑胶噪声
- **节奏**: 70-90 BPM, 摇摆感
- **调性**: 爵士和弦
- **人声**: 轻柔, 加磁带饱和
- **描述模板**: "Lo-fi, 低保真, 慵懒人声, 适合学习/放松"

### Dubstep
- **配器**: wobble bass, snare build, 撕裂音效
- **节奏**: 140 BPM, 半拍 drop
- **调性**: 小调
- **人声**: 男声, 可加 growling
- **描述模板**: "Dubstep, wobble bass, 撕裂 drop, 重低音"

### House
- **配器**: 四拍底鼓, 弦乐 pad
- **节奏**: 120-130 BPM
- **调性**: 大调
- **人声**: 女声, 偏舞曲
- **描述模板**: "House, 四拍底鼓, 舞曲, 明亮合成器"

## 2. 古典 (Classical)

### 交响 (Orchestral)
- **配器**: 弦乐群 + 木管 + 铜管 + 定音鼓
- **节奏**: 自由, 跟随旋律
- **调性**: 奏鸣曲式
- **人声**: 美声, 或纯器乐
- **描述模板**: "交响乐, 弦乐 + 木管 + 铜管, 史诗感, 大编制"

### 弦乐四重奏
- **配器**: 2 小提琴 + 中提琴 + 大提琴
- **节奏**: 自由
- **调性**: 室内乐
- **人声**: 美声/无
- **描述模板**: "弦乐四重奏, 室内乐, 优雅, 古典"

### 钢琴独奏
- **配器**: 三角钢琴为主
- **节奏**: 自由
- **调性**: 奏鸣曲/即兴
- **人声**: 美声, 或纯钢琴
- **描述模板**: "钢琴独奏, 古典浪漫派, 肖邦风格"

### 室内乐
- **配器**: 钢琴 + 弦乐 + 长笛
- **节奏**: 自由
- **描述模板**: "室内乐, 钢琴 + 弦乐 + 长笛, 精致"

## 4. 西方 (Western)

### 爵士 (Jazz)
- **配器**: 萨克斯, 钢琴, 贝斯, 鼓刷
- **节奏**: 摇摆 (swing)
- **调性**: 七和弦, 蓝调音阶
- **人声**: 爵士唱法, 转音
- **描述模板**: "爵士, 萨克斯 + 钢琴, 摇摆节奏, 转音"

### 雷鬼 (Reggae)
- **配器**: 吉他切音, 贝斯强调反拍
- **节奏**: 反拍重音
- **描述模板**: "雷鬼, 反拍重音, 慵懒人声"

### 拉丁 (Latin)
- **配器**: 康加鼓, 沙锤, 拉丁吉他
- **节奏**: 4/4 强 1-3 拍
- **描述模板**: "拉丁, 康加鼓 + 沙锤, 热情奔放"

### Bossa Nova
- **配器**: 尼龙弦吉他, 轻刷
- **节奏**: 4/4 + 切分
- **描述模板**: "Bossa Nova, 尼龙弦吉他, 慵懒"

### 蓝草 (Bluegrass)
- **配器**: 班卓琴, 曼陀林, 小提琴
- **节奏**: 快 4/4
- **描述模板**: "蓝草, 班卓琴 + 曼陀林, 快速"

---

## 5. 摇滚 (Rock)

### 摇滚 (Rock)
- **配器**: 电吉他, 失真, 鼓, 贝斯
- **节奏**: 120-140 BPM
- **人声**: 嘶吼
- **描述模板**: "摇滚, 电吉他失真, 鼓贝斯, 嘶吼人声"

### 朋克 (Punk)
- **配器**: 三和弦吉他, 快鼓
- **节奏**: 160+ BPM
- **人声**: 喊叫
- **描述模板**: "朋克, 三和弦, 极快节奏, 喊叫"

### 重金属 (Heavy Metal)
- **配器**: 失真吉他, 双底鼓
- **节奏**: 160+ BPM
- **人声**: growl, scream
- **描述模板**: "重金属, 失真吉他 + 双底鼓, growl"

### 后摇 (Post-rock)
- **配器**: 大段器乐, 渐强
- **节奏**: 自由
- **描述模板**: "后摇, 大段器乐, 渐强爆发, 无主唱"

## 6. 流行变体 (Pop Variants)

### 抒情 Ballad
- **配器**: 钢琴, 弦乐
- **节奏**: 60-80 BPM
- **人声**: 深情
- **描述模板**: "抒情 Ballad, 钢琴 + 弦乐, 深情慢板"

### R&B
- **配器**: 鼓机, 贝斯滑音, 电钢琴
- **节奏**: 90-100 BPM
- **人声**: 转音, 假声
- **描述模板**: "R&B, 鼓机 + 电钢琴, 转音假声, 律动"

### 灵魂 Soul
- **配器**: 铜管, 福音合唱
- **节奏**: 摇摆
- **人声**: 高音爆发
- **描述模板**: "灵魂乐, 铜管 + 福音合唱, 高音爆发"

### 放克 (Funk)
- **配器**: slap 贝斯, 切音吉他
- **节奏**: 强 1 拍
- **描述模板**: "放克, slap 贝斯, 切音吉他, 律动极强"

### 迪斯科 (Disco)
- **配器**: 弦乐 + 四拍底鼓
- **节奏**: 120 BPM
- **描述模板**: "迪斯科, 弦乐 + 四拍底鼓, 70 年代风"
