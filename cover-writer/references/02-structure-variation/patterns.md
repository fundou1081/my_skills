# 结构变奏模式 (Structural Variation Patterns)

## 1. 加快 (Tempo Up)

**适用**: 把抒情/慢歌改成舞曲/快歌  
**典型场景**: 民谣 → EDM, Ballad → House

**BPM 调整**:
- 原 60-80 BPM → 改 110-130 BPM
- 原 90-110 BPM → 改 130-150 BPM

**结构变化**:
- 删冗余的过渡句
- 强化 hook (副歌关键句)
- 重复主歌关键句 (制造 drop 感)

**歌词变化**:
- 删 A2 (主歌 2)
- 副歌保持不变
- 加 [intro] (4-8 拍过渡)

**minimax 输入示例**:
```
[verse] (缩短, 4 句)
[chorus]
[chorus] (重复)
[bridge] (短, 2 句)
[chorus] (重复)
```

---

## 2. 放慢 (Tempo Down)

**适用**: 把快歌改成抒情/慢歌  
**典型场景**: 摇滚 → 民谣, 流行 → Ballad

**BPM 调整**:
- 原 120+ BPM → 改 60-80 BPM
- 原 90-110 BPM → 改 60-80 BPM

**结构变化**:
- 加过渡段
- 加桥段
- 主歌可拆成 A1 + A2

**歌词变化**:
- 长句拆短 (加停顿)
- 重复主歌 (强调情感)
- 加 [outro] (渐弱)

**minimax 输入示例**:
```
[intro] (4 句, 慢起)
[verse]
[verse] (重复, 加停顿)
[chorus]
[bridge] (加 D 段)
[chorus] (重复)
[outro] (4 句, 渐弱)
```

---

## 3. 加桥 (Add Bridge)

**适用**: 增强叙事, 转折, 增强对比  
**典型场景**: 任何曲风可加

**结构变化**:
- 在 C2 之后, C3 之前插入 D 段
- D 段 4-6 句
- 主题转折或升华

**歌词生成**:
- 4-6 句桥段
- 内容: 转折/反思/升华
- 押韵: 跟前后段呼应

**minimax 输入示例**:
```
[verse]
[chorus]
[verse]
[chorus]
[bridge]   ← 新增
[chorus] (升华)
```

---

## 4. 分段 (Section Split)

**适用**: 长副歌拆成两段, 制造对比  
**典型场景**: 民谣 / 抒情, 长 hook

**结构变化**:
- 长副歌 (8+ 句) 拆成 C1a + C1b
- C1a: 主体 (前 4 句)
- C1b: 变体 (后 4 句, 改 1-2 句)

**歌词生成**:
- C1a 保持原样
- C1b 改 1-2 句, 加对比/递进

**minimax 输入示例**:
```
[verse]
[chorus] (前 4 句)
[chorus] (后 4 句, 变体)  ← 拆开
[verse]
[chorus]
```

---

## 5. 重复副歌 (Chorus Repeat)

**适用**: 强化记忆点, 制造爆发感  
**典型场景**: 流行, EDM, 摇滚

**结构变化**:
- 副歌 ×2-3
- 重复之间可加短暂 [break] 或 [bridge]

**歌词生成**:
- 副歌保持不变 (完全重复)
- 可加 [outro] 用副歌渐弱

**minimax 输入示例**:
```
[verse]
[chorus]
[chorus]  ← 重复
[chorus]  ← 再重复
[outro]
```

---

## 变奏组合 (Combined Patterns)

实际改编常**叠加多种变奏**:

### 组合 A: 加快 + 重复副歌
- 抒情 → 舞曲
- 副歌 ×3
- BPM +50%

### 组合 B: 放慢 + 加桥
- 快歌 → 抒情
- 插入 D 段
- 副歌保持

### 组合 C: 分段 + 加快
- 长副歌拆 C1a + C1b
- BPM +30%

### 组合 D: 全部叠加
- 加快 + 加桥 + 重复副歌
- (慎用, 容易乱)

## 决策流程

```
用户: "改编 + 变奏"
   │
   ├─ 问: 目标曲风? (走 01)
   │
   ├─ 问: 速度? (加快/放慢/不变)
   │
   ├─ 问: 加段? (加桥/分段/重复)
   │
   └─ 组合应用
```
