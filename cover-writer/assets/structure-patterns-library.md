# 结构模式库 (Structure Patterns Library)

> 5 种结构变奏的完整版, 含 BPM / 段落 / 歌词调整 / 适用曲风。

## 1. 加快 (Tempo Up)

### 通用规则
- BPM +30-50%
- 删冗余句 (主歌 2 / 桥段)
- 强化 hook (副歌关键句 ×2)
- 加 [intro] (4-8 拍)

### 段落模板
```
[intro] (短, 4-8 拍)
[verse] (缩短, 4 句)
[chorus]
[chorus] (重复, 强化)
[bridge] (可选, 2 句短)
[chorus] (重复)
```

### 适用曲风
- 抒情 → EDM
- 流行 → House
- 民谣 → 摇滚
- Ballad → 流行舞曲

### 不适用
- 古典 → EDM (节奏冲突)
- 爵士 → Dubstep (律动冲突)

---

## 2. 放慢 (Tempo Down)

### 通用规则
- BPM -30-50%
- 拆长句 (加停顿)
- 加 [intro] + [outro]
- 重复主歌

### 段落模板
```
[intro] (8-16 拍, 慢起)
[verse]
[verse] (重复, 加停顿)
[chorus]
[bridge] (加 D 段)
[chorus] (重复)
[outro] (4-8 句, 渐弱)
```

### 适用曲风
- 摇滚 → 民谣
- 流行 → Ballad
- EDM → Trip-hop
- 快民谣 → 慢民谣

### 不适用
- 已经是慢歌 (BPM < 60)
- 舞曲 (失律动)

---

## 3. 加桥 (Add Bridge)

### 通用规则
- 在 C2 之后插入 D 段
- D 段 4-6 句
- 主题转折/升华
- 押韵跟前后段呼应

### 段落模板
```
[verse]
[chorus]
[verse]
[chorus]
[bridge]   ← 新增
[chorus] (升华, 可改 1-2 句)
```

### 适用曲风
- 所有 (尤其流行/民谣/摇滚)

### 不适用
- 已经是大型结构 (有 D 段)

---

## 4. 分段 (Section Split)

### 通用规则
- 长副歌 (8+ 句) 拆 C1a + C1b
- C1a: 主体
- C1b: 变体, 改 1-2 句
- 加对比/递进

### 段落模板
```
[verse]
[chorus] (前 4 句)
[chorus] (后 4 句, 变体)  ← 拆开
[verse]
[chorus]
```

### 适用曲风
- 民谣 / 抒情 (长 hook)
- R&B (长副歌)

### 不适用
- 短副歌 (< 4 句)

---

## 5. 重复副歌 (Chorus Repeat)

### 通用规则
- 副歌 ×2-3
- 重复之间可加 [break] 或 [bridge] (短)
- [outro] 用副歌渐弱

### 段落模板
```
[verse]
[chorus]
[chorus]  ← 重复
[chorus]  ← 再重复
[outro] (副歌渐弱)
```

### 适用曲风
- 流行 / EDM / 摇滚
- 洗脑神曲

### 不适用
- 叙事性强的歌 (民谣)
- 已有重复的 (重复会冗余)

---

## 变奏决策流程

```
用户: "改编 + 变奏"
   │
   ├─ 问: 目标曲风? (走 01)
   │
   ├─ 问: 速度? (加快/放慢/不变)
   │      ├─ 加快 → 用模式 1
   │      └─ 放慢 → 用模式 2
   │
   ├─ 问: 加段? (加桥/分段/重复)
   │      ├─ 加桥 → 用模式 3
   │      ├─ 分段 → 用模式 4
   │      └─ 重复 → 用模式 5
   │
   └─ 组合应用 (1-2 种叠加)
```

## 禁忌

| 禁忌 | 原因 |
|---|---|
| 加快 + 重复副歌 ×3 + 分段 | 太长, 失去结构 |
| 放慢 + 加桥 + 分段 | 节奏散 |
| 全部叠加 | 一定乱 |

**原则**: **最多 2 种变奏叠加**, 保持结构清晰。
