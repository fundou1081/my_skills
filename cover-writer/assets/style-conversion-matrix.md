## 决策树

```
用户给原曲 + 目标风格
   │
   ├─ 同根源组合? (例如: 古风→民乐) → 改动小, 重点保意象
   │
   ├─ 强对比组合? (例如: 古风→EDM) → 改动大, description 是关键
   │
   └─ 升/降维组合? (例如: 民谣→交响) → 升维 (扩大编制)
                          (例如: 摇滚→民谣) → 降维 (缩减编制)
```

## 跨风格适配 (Cross-style Adaptation)

### 古风 → 电子
- **核心情感保留**: 思念/江湖/豪迈
- **配器**: 笛箫 → 合成器 lead + 古筝采样
- **节奏**: 慢板 → 128 BPM
- **人声**: 女声 → 女声 + autotune
- **特殊**: 加 808 bass, 强 drop

### 民谣 → 交响
- **核心情感保留**: 真实/朴素/孤独
- **配器**: 木吉他 → 弦乐群 + 木管
- **节奏**: 自由
- **人声**: 男声 → 男声 + 弦乐陪衬
- **特殊**: 加定音鼓, 升 key

### 流行 → 爵士
- **核心情感保留**: 爱情/怀念
- **配器**: 鼓机 → 钢琴 + 贝斯 + 鼓刷
- **节奏**: 4/4 → swing
- **人声**: 直白 → 转音, 假声
- **特殊**: 加 7 和弦, 蓝调音阶

## 风险与禁忌

| 禁忌 | 原因 | 替代 |
|---|---|---|
| 古风 → 后摇 | 完全去掉人声, 失去叙事 | 古风 → 民乐 (保留) |
| 民谣 → 重金属 | 强对比但失去民谣灵魂 | 民谣 → 交响 (升维) |
| R&B → EDM | 律动丧失 | R&B → 放克 (律动化) |
| 电子 → 民乐 | 音色冲突 | 电子 → 古典 (钢琴版) |

**原则**: 改编是"重新演绎", 不是"破坏"。保留至少 1 个核心元素 (情感/意象/旋律)。
