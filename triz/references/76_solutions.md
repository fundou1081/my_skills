# TRIZ 76个标准解（标准发明解）

> 来源：Genrich Altshuller，基于专利分析总结的物场模型标准解法

## 物场分析基础

**物场模型**：任何技术系统功能 = 物质S₁ + 物质S₂ + 场F 的三元素结构

```
S₁ ——F——> S₂
 (工具) (作用) (对象)
```

**四种系统状态**：
- **不完整系统**：缺少S₂或F
- **有害系统**：存在有害作用
- **不足系统**：功能效果不够充分
- **有效系统**：功能正常（无需改进）

## 使用流程

```
1. 构建物场模型
   ↓
2. 识别问题类型
   ├─ 不完整系统 → 第1级
   ├─ 有害系统   → 第1级
   ├─ 不足系统   → 第2级
   └─ 测量问题   → 第4级
   ↓
3. 选择对应标准解
   ↓
4. 结合分离原理（如需要）
   ↓
5. 转化为具体方案
```

## 物场模型符号

```
完整有用系统:   S₁—F—>S₂

不完整系统:     S₁—?   (缺少场)
               ?—F—>S₂ (缺少工具)

有害系统:       S₁—F—X S₂  (有害作用)

测量系统:       S₁—F—>S₂
                ↓检测
                S₃感知
```

## 分离原理（用于物理矛盾）

| 分离类型 | 应用场景 | 典型解法 |
|----------|----------|----------|
| **空间分离** | 在不同空间位置满足相反需求 | 分区设计、多区域结构 |
| **时间分离** | 在不同时时间段满足相反需求 | 先后执行、交替进行 |
| **条件分离** | 在不同条件下满足相反需求 | 条件触发、环境切换 |
| **整体与部分分离** | 整体和部分满足相反需求 | 局部差异化、自适应结构 |
