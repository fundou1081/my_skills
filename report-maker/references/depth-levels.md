# 调研深度等级指南

## L1 - 快速概览

### 适用场景
- 初步了解问题
- 快速决策支持
- 时间紧迫的情况

### 调研特点
- **信息来源**：权威综述、主要媒体报道
- **专家数量**：2-3个核心专家
- **分析方法**：单一核心方法
- **数据深度**：宏观数据、主要指标

### 时间投入
- 分析时间：1-2小时
- 报告篇幅：3-5页 / 1500-2500字

### 输出重点
```
✓ 核心结论清晰
✓ 关键数据呈现
✓ 主要建议明确
✗ 深度论证有限
✗ 细节分析简化
```

### 典型问题示例
- "快速分析一下XXX行业"
- "给我一个XX问题的概览"
- "简单评估一下这个方案"

---

## L2 - 标准调研

### 适用场景
- 常规决策支持
- 方案评估
- 中等复杂度问题

### 调研特点
- **信息来源**：行业报告、专业媒体、官方数据
- **专家数量**：4-5个专家
- **分析方法**：2种分析方法组合
- **数据深度**：行业数据、细分数据

### 时间投入
- 分析时间：4-8小时
- 报告篇幅：8-15页 / 4000-8000字

### 输出重点
```
✓ 结论有数据支撑
✓ 多角度分析
✓ 建议可操作性
✓ 风险识别
△ 细节分析适中
```

### 典型问题示例
- "分析一下XXX市场竞争格局"
- "评估这个投资机会"
- "制定XX问题的解决方案"

---

## L3 - 深度调研

### 适用场景
- 重要决策支持
- 战略规划
- 复杂问题诊断

### 调研特点
- **信息来源**：多源数据、专业报告、实地调研
- **专家数量**：5-6个专家
- **分析方法**：多种方法综合运用
- **数据深度**：详尽数据、交叉验证

### 时间投入
- 分析时间：1-3天
- 报告篇幅：20-40页 / 10000-20000字

### 输出重点
```
✓ 深入全面的分析
✓ 多源数据交叉验证
✓ 完整的论证链条
✓ 详细的实施方案
✓ 全面的风险评估
```

### 典型问题示例
- "深入分析XXX战略方向"
- "全面诊断XX业务问题"
- "制定XX转型的详细方案"

---

## L4 - 专家级

### 适用场景
- 学术研究
- 政策制定
- 重大投资决策
- 专业领域深度研究

### 调研特点
- **信息来源**：学术文献、一手数据、专家访谈
- **专家数量**：6-7个专家
- **分析方法**：学术级方法论、原创分析框架
- **数据深度**：原始数据、定制研究

### 时间投入
- 分析时间：1-2周
- 报告篇幅：50+页 / 25000+字

### 输出重点
```
✓ 学术级别的严谨性
✓ 原创分析框架
✓ 完整的文献引用
✓ 可复现的方法论
✓ 对现有研究的贡献
```

### 典型问题示例
- "开展XX领域的学术研究"
- "为XX政策制定提供依据"
- "对XX进行专家级评估"

## 等级对比表

| 维度 | L1 快速概览 | L2 标准调研 | L3 深度调研 | L4 专家级 |
|-----|-----------|-----------|-----------|----------|
| 分析时间 | 1-2小时 | 4-8小时 | 1-3天 | 1-2周 |
| 报告篇幅 | 3-5页 | 8-15页 | 20-40页 | 50+页 |
| 专家数量 | 2-3个 | 4-5个 | 5-6个 | 6-7个 |
| 分析方法 | 1种 | 2种 | 多种 | 综合框架 |
| 数据来源 | 二手综述 | 行业报告 | 多源数据 | 原始数据 |
| 论证深度 | 结论导向 | 数据支撑 | 完整链条 | 学术级别 |
| 适用场景 | 快速决策 | 常规决策 | 重要决策 | 学术/政策 |

---

## 深度选择建议

### 时间紧迫程度

```
< 1天  → L1 或 L2
1-3天  → L2 或 L3
1周+   → L3 或 L4
```

### 决策重要程度

```
常规决策  → L1 或 L2
重要决策  → L2 或 L3
重大决策  → L3 或 L4
```

### 问题复杂程度

```
简单问题  → L1
中等复杂  → L2
复杂问题  → L3
极复杂    → L4
```

### 受众专业程度

```
一般大众  → L1 或 L2
专业人士  → L2 或 L3
专家学者  → L3 或 L4
```

---

## 注意事项

1. **用户意愿优先**：用户指定的深度等级为最终决定
2. **资源匹配**：确保有足够时间/资源完成所选深度的调研
3. **灵活调整**：调研过程中可根据发现调整深度
4. **透明沟通**：如需调整深度，与用户沟通确认
5. **质量优先**：宁可降低深度保证质量，不要深度虚高内容空洞
